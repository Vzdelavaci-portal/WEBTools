# 📋 Mini Kanban Board

A modern Kanban Board built with:

- HTML
- CSS
- JavaScript

Organize your tasks with drag & drop, priorities, labels, due dates and automatic local storage.

---

# ✨ Features

✅ Drag & Drop Tasks  
✅ Three Kanban Columns  
✅ Task Priorities  
✅ Color Labels  
✅ Due Dates  
✅ Search Tasks  
✅ Label Filtering  
✅ Checklist Support  
✅ Progress Statistics  
✅ Progress Bar  
✅ LocalStorage  
✅ Export JSON  
✅ Import JSON  
✅ Confetti Celebration 🎉  
✅ Responsive Design

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

No frameworks.

No backend.

Works completely offline.

---

# 📸 Features Overview

The board allows you to:

- create tasks
- edit tasks
- drag tasks between columns
- organize work visually
- track project progress
- export and import your board
- automatically save everything

---

# 📋 Kanban Columns

Three workflow stages:

```text
📋 To Do

⚡ In Progress

✅ Done
```

Simply drag a task between columns.

---

# 🖱 Drag & Drop

- project management
- personal planning
- school work
- software development
