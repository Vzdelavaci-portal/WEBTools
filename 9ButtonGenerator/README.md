# 🎨 CSS Button Generator

A modern CSS button generator built with:

- HTML
- CSS
- JavaScript

Create beautiful buttons with live preview, customizable styles and instant CSS export.

# 💡 Example CSS Output

```css
.button {
  background: #2563eb;
  color: #ffffff;
  padding: 14px 28px;
  font-size: 16px;
  font-weight: 900;
  border: none;
  border-radius: 14px;
  box-shadow: 0 12px 30px rgba(37, 99, 235, 0.35);
  cursor: pointer;
  transition: 0.25s ease;
}

.button:hover {
  transform: translateY(-4px) scale(1.03);
}
```
