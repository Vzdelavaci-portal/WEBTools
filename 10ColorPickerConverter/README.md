# 🎨 Color Picker & Converter

A modern color tool built with:

- HTML
- CSS
- JavaScript

Pick colors, convert between color formats, generate palettes and check accessibility contrast ratios in real time.

---

# ✨ Features

✅ Live color preview  
✅ HEX color input  
✅ Color picker input  
✅ HEX to RGB conversion  
✅ HEX to HSL conversion  
✅ HEX to HSV conversion  
✅ HEX to CMYK conversion  
✅ Generated color palette  
✅ Contrast checker  
✅ Gradient generator preview  
✅ Random color generator  
✅ Copy values to clipboard  
✅ Responsive modern UI

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

No frameworks.  
No libraries.  
Pure frontend development.

---

# 📸 Features Overview

The tool allows you to:

- pick any color visually
- convert colors between formats
- generate lighter and darker variations
- build matching color palettes
- check accessibility contrast
- create gradients
- copy color values instantly
- experiment with color design

# ♿ Contrast Checker

Accessibility preview for:

### White Text

```text
AAA Pass
AA Pass
Fail
```

### Black Text

```text
AAA Pass
AA Pass
Fail
```

Includes real contrast ratio calculations.

Example:

```text
7.42 : 1
```

Useful for:

- accessibility testing
- UI design
- WCAG compliance

---

# 🌈 Gradient Preview
