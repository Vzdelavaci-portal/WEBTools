- HTML
- CSS
- JavaScript

Create beautiful animated loading indicators with live preview, customizable settings and instant HTML/CSS export.

# 💡 Example CSS Output

```css
.loader {
  --loader-size: 80px;
  --loader-color: #38bdf8;
  --loader-speed: 1s;
  --loader-thickness: 8px;

  width: var(--loader-size);
  height: var(--loader-size);
  color: var(--loader-color);
}

.loader-spinner {
  border: var(--loader-thickness) solid currentColor;
  border-color: currentColor transparent currentColor transparent;
  border-radius: 50%;
  animation: loader-spin var(--loader-speed) linear infinite;
}
```
