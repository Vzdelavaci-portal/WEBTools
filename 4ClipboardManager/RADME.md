# 📋 Clipboard Manager

A modern clipboard and notes manager built with:

- HTML
- CSS
- JavaScript

Save, manage and quickly copy your text snippets, notes and clipboard content directly in your browser.

---

# ✨ Features

✅ Save clipboard notes  
✅ Copy notes instantly  
✅ Delete individual notes  
✅ Delete all notes  
✅ Responsive modern UI  
✅ LocalStorage support  
✅ Persistent saved notes  
✅ Pure HTML/CSS/JS (no libraries)

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

No frameworks.  
No libraries.  
Just pure frontend development.

---

# 📸 Features Overview

The app allows you to:

- save important text snippets
- manage clipboard history
- copy saved notes instantly
- store notes in browser local storage
- organize quick reusable text
- use the app offline
