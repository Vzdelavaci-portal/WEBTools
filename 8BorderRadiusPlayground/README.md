# 🎛️ Border Radius Playground

A modern CSS border-radius generator built with:

- HTML
- CSS
- JavaScript

Create rounded corners, pills, circles and organic blob shapes visually with live preview and instant CSS generation.

---

# ✨ Features

✅ Live border-radius preview  
✅ Individual corner controls  
✅ Blob shape generator  
✅ Random shape generator  
✅ CSS code generation  
✅ One-click CSS copy  
✅ Preset shapes  
✅ Responsive modern UI  
✅ Pure HTML/CSS/JS (no libraries)

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

No frameworks.  
No libraries.  
Just pure frontend development.

---

# 📸 Features Overview

The generator allows you to:

- customize each corner independently
- create rounded rectangles
- generate circles
- create pill-shaped buttons
- design organic blob shapes
- generate random border-radius combinations
- copy CSS instantly
- experiment with modern UI designs

---

# 🎨 Available Modes

## Basic Corners

Control each corner separately:

```css
border-radius: 20px 60px 10px 80px;
```

Adjust:

- Top Left
- Top Right
- Bottom Right
- Bottom Left

---

## Blob Mode

Create organic and fluid shapes using percentage values.

Example:

```css
border-radius:
  62% 38% 58% 42% /
  42% 58% 38% 62%;
```

Perfect for:

- landing pages
- SaaS websites
- modern UI elements
- creative backgrounds
