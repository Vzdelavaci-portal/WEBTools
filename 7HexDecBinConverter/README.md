# 🔢 HEX DEC BIN Converter

A modern number system converter built with:

- HTML
- CSS
- JavaScript

Convert values between Decimal, Binary, Hexadecimal and Octal instantly with a live bit visualization.

---

# ✨ Features

✅ Decimal to Binary conversion  
✅ Decimal to Hexadecimal conversion  
✅ Decimal to Octal conversion  
✅ Binary to Decimal conversion  
✅ Hexadecimal to Decimal conversion  
✅ Octal to Decimal conversion  
✅ Live conversion updates  
✅ Bit visualization (8, 16, 32 bit)  
✅ Copy results to clipboard  
✅ Quick examples  
✅ Responsive modern UI

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

No frameworks.  
No libraries.  
Just pure frontend development.

---

# 📸 Features Overview

The converter allows you to:

- convert between number systems instantly
- visualize binary bits
- inspect bit positions
- copy generated values
- test hexadecimal values
- explore binary representation
- learn how different number systems work

---

# 🔢 Supported Number Systems

### Decimal (DEC)

```text
255
1024
65535
```

### Binary (BIN)

```text
11111111
10101010
10000000000
```

### Hexadecimal (HEX)

```text
FF
1A3
FFFF
```

### Octal (OCT)

```text
377
2000
177777
```

---

# 🧠 Bit Viewer

The Bit Viewer provides:

- 8-bit mode
- 16-bit mode
- 32-bit mode

Example:

```text
255

11111111
```

Each bit position is displayed individually.
