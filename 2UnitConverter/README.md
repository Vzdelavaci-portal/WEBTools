# 🧮 Advanced Unit Converter

A modern and responsive unit converter built with:

- HTML
- CSS
- JavaScript

Convert between dozens of units instantly with a clean UI and live conversion results.

---

# ✨ Features

✅ Live conversion results  
✅ Multiple categories  
✅ Responsive modern design  
✅ Easy-to-use interface  
✅ Unit swapping  
✅ Quick value buttons  
✅ Pure HTML/CSS/JS (no libraries)
