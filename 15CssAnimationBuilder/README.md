# 🎬 CSS Animation Builder

A modern CSS Animation Builder built with:

- HTML
- CSS
- JavaScript

Create beautiful CSS animations visually, preview them in real time and export ready-to-use code.

---

# ✨ Features

✅ Live animation preview  
✅ Multiple animation types  
✅ Multiple animated objects  
✅ Color customization  
✅ Duration control  
✅ Delay control  
✅ Iteration settings  
✅ Timing function selection  
✅ Direction control  
✅ Random animation generator  
✅ Replay animation  
✅ CSS export  
✅ Responsive design

---

- HTML5
- CSS3

The tool allows you to:

# 🎭 Available Animations

## Fade In

```css
opacity: 0 → 1;
```

Smooth fade appearance.

---

## Slide Up

```css
translateY(120px) → 0;
```

Element enters from below.

---

## Slide Down

```css
translateY(-120px) → 0;
```

Element enters from above.

---

## Slide Left

```css
translateX(120px) → 0;
```

Element enters from the right.

---

## Slide Right

```css
translateX(-120px) → 0;
```

Element enters from the left.

---

## Scale In

```css
scale(0.3) → scale(1);
```

Smooth zoom effect.

---

## Rotate

```css
rotate(360deg);
```

Full rotation animation.

---

## Bounce

```css
translateY(...)
```

Bouncing movement.

---

## Pulse

```css
scale(...)
```

Growing and shrinking effect.

---

## Shake

```css
translateX(...)
```

Horizontal shaking.

---

## Flip

```css
rotateY(...)
```

3D flip animation.

---

## Swing

```css
rotate(...)
```

Pendulum-like movement.

# 📋 CSS Export

One-click export:

```css
.animated-object {
  animation-name: bounce;
  animation-duration: 1.2s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;
}
```

Ready for:

- websites
- landing pages
- dashboards
- SaaS products
- personal projects

---

# 💡 Example Generated CSS

```css
.animated-object {
  animation-name: pulse;
  animation-duration: 1.5s;
  animation-delay: 0s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}
```
