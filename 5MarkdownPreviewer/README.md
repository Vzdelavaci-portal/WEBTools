# 📝 Markdown Previewer

A modern live Markdown editor and previewer built with:

- HTML
- CSS
- JavaScript

Write Markdown on the left and instantly preview the rendered result on the right.

---

## ✨ Features

✅ Live Markdown preview  
✅ Real-time rendering  
✅ Responsive modern UI  
✅ Markdown formatting support  
✅ Code block support  
✅ Links and lists support  
✅ Clear editor button  
✅ Pure frontend project

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Marked.js CDN

No backend required.  
Runs entirely in the browser.

---

## 📸 Supported Markdown Features

The previewer supports:

- headings
- bold text
- italic text
- lists
- links
- images
- code blocks
- quotes
- tables
- task lists
- inline code
