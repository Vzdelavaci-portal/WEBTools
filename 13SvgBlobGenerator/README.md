# 🫧 Blob Parameters

## Points

Controls how many points the blob contains.

```text
4 → Simple Shape
8 → Balanced Blob
16 → Complex Blob
```

Higher values create more detailed shapes.

---

## Irregularity

Controls randomness of the blob.

```text
0%   → Almost Circle
35%  → Natural Blob
70%  → Very Organic
```

---

## Smoothness

Controls curve smoothing.

```text
20%  → Sharp Curves
65%  → Balanced Curves
100% → Very Smooth Blob
```

# 💡 Example SVG Output

```svg
<svg viewBox="0 0 500 500">

  <defs>
    <linearGradient
      id="blobGradient"
      x1="0%"
      y1="0%"
      x2="100%"
      y2="100%"
    >
      <stop offset="0%" stop-color="#38BDF8" />
      <stop offset="100%" stop-color="#8B5CF6" />
    </linearGradient>
  </defs>

  <path
    d="M ..."
    fill="url(#blobGradient)"
  />

</svg>
```
