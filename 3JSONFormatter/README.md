# 🧩 JSON Formatter

A modern and responsive JSON formatter built with:

- HTML
- CSS
- JavaScript

Format, validate and beautify JSON instantly with live error detection and line highlighting.

---

# ✨ Features

✅ JSON formatting  
✅ JSON minification  
✅ Live JSON validation  
✅ Error line highlighting  
✅ Error line and column detection  
✅ Copy formatted JSON  
✅ Paste from clipboard  
✅ Responsive modern UI  
✅ Pure HTML/CSS/JS (no libraries)

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

No frameworks.  
No libraries.  
Just pure frontend development.

---

# 📸 Features Overview

The formatter allows you to:

- validate JSON instantly
- detect invalid JSON
- highlight lines with errors
- format JSON for readability
- minify JSON for production
- copy formatted output
- paste directly from clipboard
