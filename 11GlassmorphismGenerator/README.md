# 🌫 Blur Control

Adjust the glass blur effect:

```css
backdrop-filter: blur(18px);
```

Examples:

```text
0px   → No Blur
10px  → Light Blur
20px  → Medium Blur
40px  → Strong Blur
50px  → Extreme Blur
```

---

# 💎 Transparency Control

Adjust glass opacity:

```css
background: rgba(255,255,255,0.18);
```

Examples:

```text
0.05 → Very Transparent
0.18 → Modern Glass
0.30 → Frosted Glass
0.50 → Strong Glass
```

---

# 🔲 Border Radius

Create different card styles:

```css
border-radius: 28px;
```

Examples:

```text
0px  → Sharp Corners
16px → Modern Card
28px → Soft Card
50px → Highly Rounded
```

---

# ✨ Border Settings

Generate glass borders:

```css
border: 1px solid rgba(255,255,255,0.28);
```

Control:

- Border Opacity
- Light Border Toggle
