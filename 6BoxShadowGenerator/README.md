# 🌑 Box Shadow Generator

A modern CSS box-shadow generator built with:

- HTML
- CSS
- JavaScript

Create beautiful box shadows visually and instantly copy the generated CSS code.

---

# ✨ Features

✅ Live box-shadow preview  
✅ Real-time shadow generation  
✅ Responsive modern UI  
✅ Copy CSS button  
✅ Shadow color picker  
✅ Opacity control  
✅ Inset shadow support  
✅ Reset generator button  
✅ Pure HTML/CSS/JS (no libraries)

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

No frameworks.  
No libraries.  
Just pure frontend development.

---

# 📸 Features Overview

The generator allows you to:

- adjust horizontal shadow offset
- adjust vertical shadow offset
- change blur radius
- change spread radius
- control shadow opacity
- pick custom shadow colors
- generate inset shadows
- copy generated CSS instantly

---

# ▶️ How to Run

Simply open:

```text
index.html
```

in your browser.

No installation required.

---

# 💡 Example Generated CSS

```css
box-shadow: 10px 10px 30px 0px rgba(0, 0, 0, 0.35);
```
