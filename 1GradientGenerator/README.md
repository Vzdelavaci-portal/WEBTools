# 🎨 Gradient Generator

A simple and modern gradient generator built with **HTML, CSS and JavaScript**.

Create beautiful CSS gradients, preview them instantly and copy the generated CSS code with one click.

---

# ✨ Features

✅ Live gradient preview  
✅ Multiple gradient directions  
✅ Random gradient generator  
✅ Copy CSS code to clipboard  
✅ Responsive design  
✅ Glassmorphism UI style  
✅ Pure HTML/CSS/JS (no libraries)

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

# 📸 Preview

The app allows you to:

- choose two colors
- change gradient direction
- generate random gradients
- instantly copy the CSS code
